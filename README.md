## • Sobre mim

Sou estudante de Engenharia da Computação com foco no desenvolvimento de sistemas seguros, escaláveis e eficientes. Minha experiência abrange desde a arquitetura de sistemas com criptografia de ponta a ponta até o desenvolvimento de aplicações móveis e sistemas inteligentes.

Priorizo a escrita de código limpo, o uso de padrões de projeto e uma abordagem proativa para resolver desafios técnicos complexos.
